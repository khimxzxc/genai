# Система Автоматизированного Code Review (AI Code Evaluator)

Это полноценное Fullstack-приложение для автоматизированной проверки и оценки студенческих работ (HTML/CSS) с использованием мультимодальной модели **Google Gemini 2.5 Flash**. Система анализирует код, делает скриншоты результата через Puppeteer и предоставляет детальную обратную связь с оценкой по 100-балльной шкале.

## 🚀 Архитектура проекта

- **Frontend:** Vite + React (UI для студентов и преподавателей).
- **Backend:** Node.js + Express.js (Бизнес-логика, создание скриншотов, работа с AI).
- **База данных:** PostgreSQL (через **Prisma ORM**).
- **AI Интеграция:** Google Gemini 2.5 Flash API (для высокоточного анализа с использованием RAG и Chain-of-Thought).
- **Рендеринг:** Puppeteer (для снятия скриншотов студенческих работ на бэкенде).

## 📂 Структура репозитория

- `/frontend` — React приложение (Панель студента, Панель преподавателя, интерфейс загрузки проектов).
- `/backend` — Express.js сервер (API маршруты, Prisma ORM, интеграция с Gemini, Puppeteer).

## 🛠 Установка и запуск (Локально)

### Предварительные требования
- Node.js (v18+)
- База данных PostgreSQL
- API ключ от Google Gemini

### 1. Клонирование репозитория
```bash
git clone <url_репозитория>
cd <папка_проекта>
```

### 2. Настройка Backend
```bash
cd backend
npm install
```
Создайте файл `.env` в папке `backend` по примеру:
```env
PORT=3000
DATABASE_URL=postgresql://user:password@localhost:5432/dbname
GEMINI_API_KEY=ваш_api_ключ_gemini
JWT_SECRET=ваш_секретный_ключ
```
Примените миграции базы данных:
```bash
npx prisma migrate dev
```
Запуск сервера:
```bash
npm run dev
```

### 3. Настройка Frontend
```bash
cd ../frontend
npm install
```
Создайте файл `.env` в папке `frontend` по примеру:
```env
VITE_API_URL=http://localhost:3000/api
```
Запуск клиента:
```bash
npm run dev
```

## 🧠 Ключевые особенности
- **Оценка по 3 критериям:** HTML (30 баллов), CSS (30 баллов), UI/UX (40 баллов).
- **Продвинутый AI-анализ:** Использование Google Gemini 2.5 Flash с методологиями RAG и Chain-of-Thought для минимизации галлюцинаций.
- **Интернационализация (i18n):** Вся обратная связь генерируется ИИ строго на **казахском языке** (в соответствии с академическими стандартами).
- **Ролевая система:** Поддержка JWT аутентификации для разделения прав доступа (Студент / Преподаватель).
- **История сабмитов:** Сохранение всех проверок в PostgreSQL (Prisma) для дальнейшего анализа преподавателем.
